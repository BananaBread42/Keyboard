# Keycodes Viewer

A simple and interactive website that displays keyboard event information whenever a key is pressed.


## Example

Pressing the `A` key shows:

`e.key` → `a`
`e.code` → `KeyA`
`e.which` → `65`
`e.keyCode` → `65`
